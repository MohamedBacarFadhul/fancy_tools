# fancy_tools
## Mode de fonctionnement
### Etape 1:
clonner le ficher git sur votre repértoir dans votre local

```sh
git clone https://github.com/MohamedBacarFadhul/fancy_tools.git

```
### Etape 2:
Rendez-vous dans le dossier :
```sh
/fancy_tools/install.sh
```
### Etape 3:
executer le fichier install.sh
```sh
>>./install.sh
```
 ### Remarque :
si vous n'avaez pas la permission d'exécution utiliser la commander 
```sh
>>chmod 750 install.sh
```
### Etape 4 :
Utiliser le fichier updateFancyTools pour mettre à jour les outils fancyTools pour l'executer vous devrez lui donner les droit d'execution 
en ulisant la commande chmod comme précedent.
